# Как создать шаблон документа

Библиотека `TemplateEngine.Docx` использует **Content Controls** в Word для подстановки данных.

## Включение вкладки "Разработчик"

1. Открой Word
2. **Файл** → **Параметры** → **Настроить ленту**
3. В правом списке поставь галочку **Разработчик**
4. Нажми **OK**

## Добавление метки в шаблон

1. Поставь курсор в нужное место документа
2. Перейди на вкладку **Разработчик**
3. В группе **Элементы управления** нажми **Обычный текст** (иконка `Aa`)
4. Выдели добавленный элемент управления
5. Нажми кнопку **Свойства** (на вкладке Разработчик)
6. В поле **Тег** введи имя метки (например: `НомерДоговора`)
7. Нажми **OK**

## Пример

Для текста "Договор № _____ от _____" нужно:

1. Написать: `Договор № `
2. Вставить Content Control с тегом `НомерДоговора`
3. Написать: ` от `
4. Вставить Content Control с тегом `ДатаДоговора`
