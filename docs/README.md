# АСУГАК - Автоматизированная Система Управления ГАК

## Описание

АСУГАК - это десктопное WPF-приложение для управления процессом защиты выпускных квалификационных работ (ВКР) в рамках Государственной Аттестационной Комиссии (ГАК).

Система позволяет:
- Управлять кафедрами и сотрудниками
- Формировать составы ГАК
- Планировать заседания и периоды защит
- Вести учёт дипломников и их работ
- Управлять направлениями и профилями подготовки
- Учитывать оплату членам комиссии
- Хранить документы

## Технологический стек

| Компонент | Технология |
|-----------|------------|
| Платформа | .NET 8.0 |
| UI Framework | WPF (Windows Presentation Foundation) |
| ORM | Entity Framework Core 9.0 |
| База данных | SQLite |
| DI Container | Microsoft.Extensions.DependencyInjection |
| Архитектура | Vertical Slice Architecture (VSA) |

## Быстрый старт

### Требования

- .NET 8.0 SDK
- Windows 10/11

### Установка и запуск

```bash
# Клонировать репозиторий
git clone <repository-url>
cd asugaksharp2

# Восстановить зависимости и собрать
dotnet build

# Запустить приложение
dotnet run
```

### Работа с базой данных

```bash
# Создать миграцию
dotnet ef migrations add <MigrationName>

# Применить миграции
dotnet ef database update

# Откатить миграцию
dotnet ef database update <PreviousMigrationName>
```

## Структура проекта

```
asugaksharp2/
├── Core/                    # Доменный слой
│   └── Entities/           # Сущности (Kafedra, Person, Gak, ...)

│   ├── Diplomnik/         # CRUD для дипломников

│   └── Persistence/       # EF Core DbContext
├── docs/                   # Документация
├── App.xaml(.cs)          # Точка входа, DI конфигурация
└── MainWindows.xaml(.cs)  # Главное окно
```
