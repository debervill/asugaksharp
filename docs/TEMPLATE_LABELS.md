# Метки для шаблона договора

Используйте в docx-файле как `{{Метка}}`

## Общие данные договора

| Метка | Описание |
|-------|----------|
| `{{НомерДоговора}}` | Номер договора |
| `{{ДатаДень}}` | День даты |
| `{{ДатаМесяц}}` | Месяц даты |
| `{{ДатаГод}}` | Год даты |
| `{{ДатаДоговора}}` | Полная дата договора |
